軟體工程師扶持計畫
個人簡介
  • 過往主要工作經歷為硬體製造業的研發工程師以及專案管理，因此具備研究精神與開發思維
  • 因職涯發展想要轉職到軟體產業，這幾年從無開始接觸並了解軟體相關知識與技能
  • 在今年 2020 年完成資工系碩專班，以及就職軟體工程師，主要利用 Python 處理醫療資料
  • 希望能補足網頁相關的開發技能，具體呈現出所做的成果與產品
為什麼想成為工程師？
  • 想在 40 而不惑的時候，確實掌握一門技術，讓我能夠充滿自信並且能夠支撐人生後半場
  • 目標是以後端為主要，進一步至全端工程師為目標
  • 希望能透過自己所擁有的技術，幫助別人與社會
為了成為工程師，曾經做過什麼努力？
  • 報名不少線上課程以及購買書籍，去讀資工碩專班以及參加鐵人賽，以及剛完成的學米實戰營作品
  • 鐵人賽
  • 學米實戰營作品
  • 當然還有這篇因剛學習而產出的介紹網頁
對於工程師的職業想像是什麼？
  • 具有一個可以帶著走的技術，透過自身技術可以幫助別人或社會解決問題
如果參與這個計畫，會怎麼安排學習時間？
  • 目前預計在每週三至五的下班時段以及週六、日時段進行學習
  • 上班空檔如午休，補一些進度或是排除一些學習上遇到的問題
  • 每週一、三、五彙整進度，週日盤點回顧與預排計畫
如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？
  • 主要就是時間不夠跟遇到問題解不出來的問題
  • 但時間其實永遠不夠，所以要先規劃，讓自己在有限時間內提出當下最好的結果，而不要糾結在問題上
  • 遇到問題如果真的連搜尋都沒有方向，會先進行詢問後再思考如何拆解問題
  • 透過導師團隊的指導，讓我可以確實前進
想要對我們說的事情？
  • 先讓我說聲：「辛苦了，感謝」，願意餐與培訓教育的人我都很佩服
  • 已經可以想像這是一段不輕鬆的路，但我現在不做，以後更沒機會了
  • 雖然我在距離與經歷上都離各位比較遠，但我已經在這條路上了
  • 何況又是我所期望的 Python 領域，非常希望能加入學習的行列
  • 期望透過有系統的帶領，可以讓我少走不少冤枉路以及減少摸索的方向
  • 所列舉的項目內容都是我想學習的，能在生日前知道這個消息真是我最大的驚喜
  • 未來也希望結束後成為能夠幫助別人的那個人
